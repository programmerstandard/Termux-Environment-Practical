# Termux-Environment-Practical

Useful Tutorial About Termux

Termux combines powerful terminal emulation with an extensive Linux package
collection.

To read internal command list help, type:

```
help
```

* More tutorial to use help, read on `Foundation/UsingHelp.md`.
* For a complete reference, check the __Termux Wiki__, __Termux Handbook__, etc.

## Notes

* Google Play Store versions of Termux are restricted and may not support full functionality. Use Termux from F-Droid instead.
* Remember this repository only for development and maintenance servers!
* You may found more updated informations and tutorials in other sources.
