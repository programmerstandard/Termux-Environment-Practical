To know $PREFIX path, read in most-common-path.md

### apache path

```bash
$PREFIX/etc/apache2/
```

Configuration file that needs to check or backup with extension *.conf.

### tigervnc path

```bash
$PREFIX/var/lib/proot-distro/installed-rootfs/ubuntu/root/.config/tigervnc/
```

### vnc path

to access vnc Configuration under proot package distro go to this following path.

```bash
$PREFIX/var/lib/proot-distro/installed-rootfs/ubuntu/root/.vnc/
```

Check or backup `xstartup` file.
