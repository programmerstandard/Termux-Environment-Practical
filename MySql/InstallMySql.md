## Install Package MySql ( "mariadb" ):

```
pkg install mariadb
```

## Check Installed Package

```
 apt list — manual-installed
`

### Locationocation

```
/data/data/com.termux/files/usr/var/lib/mysql
```

## Run MySqmysqld
mysqld_safe --skip-grant-tables
```

## Login To MySql

```
mysql -u root
```

If have a password:
```
mysql -u root -p
```
